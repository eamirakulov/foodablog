== Description ==


This amazing plugin will import all of your HubSpot blog posts to WordPress blog, under one click. You just have to get HubSpot API from this link : https://app.hubspot.com/keys/get
<br />
The key features of this plugin is :  
<ul>
	<li>Setting Featured Images by downloading them in WordPress.</li>
	<li>Assign categories to posts by converting.</li>
	<li>Convert topics to categories.</li>
	<li>Prevent Dupication of posts.</li>
	<li>Use AJAX to import all the data.</li>
</ul>


== Installation ==

1. Upload the entire folder to the /wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to "HubSpot Blog To WordPress" from left menu.
4. Configure the settings by putting the API key and number of posts. API key need to be get from HubSpot


== Frequently Asked Questions ==

= What does this plugin do with "topics"? =

Topics will be converted to categories and will get assigned to respective posts.
